전자부품 제조 MES 고도화

개요

전자부품 제조 운영 과정에서 생산, 재고, 품질, 영업, 출하 데이터가 이어지는 흐름을 정리하고 서버 집계 구조를 개선한 프로젝트입니다.

역할

  • 서버 로직 및 데이터 흐름 개선
  • 집계 구조와 조회 성능 개선

기술

  • Java, Spring Boot, MySQL
  • MES, 월 마감, 집계 데이터

README 작성 뼈대

  • 문제 상황: 누적 데이터 증가로 인한 조회 성능 저하
  • 해결 방식: 월 마감 기준 집계 결과 저장 및 재계산 제거
  • 검증 포인트: 마감 이후 데이터 변경이 집계 결과에 미치는 영향 차단
  • 결과: 주요 조회 응답 시간 개선

  • LOT, 품질 검사, 상태 전이

README 작성 뼈대

  • 문제 상황: RFID 수신 순서와 실제 공정 흐름의 차이
  • 해결 방식: 순서 변경, 품목 전환 등 예외 시나리오를 서버 로직에 반영
  • 검증 포인트: 자재와 상품의 서로 다른 LOT 관리 기준 분리
